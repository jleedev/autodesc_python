AutoDesc - Wikidata automated description API

This API can describe most Wikidata items automatically. For some item types (e.g. biographies) in some languages, long descriptions are available. Blog entry, code repo, translations.

Q   Language

Mode description (where possible)

Links

Redlinks For HTML/Wikipedia links with no corresponding Wikipedia page; long desc only

Format

Infobox (for wikitext output only)

Examples:

  • Long English description of J.S.Bach, as a HTML page linking to English Wikipedia, and en.wp redlinks to Reasonator
  • Wanderer above the Sea of Fog as wikitext, with {{Infobox artwork}}
